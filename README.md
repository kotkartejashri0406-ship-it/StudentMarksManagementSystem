***Student Marks Manager System***

****Description****
The Student Marks Manager System is a Python program that allows teachers or administrators to efficiently manage student grades. 
It collects marks for multiple students and generates a detailed academic report, including average, highest, lowest marks, pass/fail count, grade distribution, and topper information. 
The system also tracks how many students scored above or below the class average.
